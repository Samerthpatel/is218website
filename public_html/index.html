Git Commands

Commands Definition
git init initialize an existing directory as a Git repository
git add [file] add a file as it looks now to your next commit (stage) / . adds all files
git commit -m “[descriptive message]” commit your staged content as a new commit snapshot
git checkout -b [branch-name] creates a branch and goes to the branch
git checkout [branch-name] goes to the specified branch
git branch list your branches. a * will appear next to the currently active branch
git push origin [branch] transmits local branch commit to the remote repository branch
git pull origin [branch] fetch and merge any commits from tracking remote branch
git clone [URL] retrieve and entire repository from a hosted location via [URL]
git merge [branch] merge the specified branch's history into the current one

Terminal Commands

Commands Definition
pwd shows present working directory
ls lists all files in the directory
cd [directory] change to the specfied directory. [..] goes back a directory, each . is a directory
mkdir [directory] create a new directory
mv [filename] [newfilename] rename a file to a new file name
touch [filename] create the specified files
vi opens a visual editor in terminal and lets you edit the file

Docker Commands

Commands Definition
Docker compose up aggregates the output of each container
Docker compose down stop all containers
Docker ps lists the running containers
Docker kill [container-id] shuts the specified container off
